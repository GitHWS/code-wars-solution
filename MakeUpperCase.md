# MakeUpperCase

## Requirement

<p>Write a function which converts the input string to uppercase.</p>

  1. 대문자로 변환하는 함수를 완성하라

## Example

```js
"hello" => "HELLO"
```

<br>

### 첫번째 나의 시도

> why? 문자열을 대문자화하는 메서드인 "toUpperCass()"를 사용하면 된다.

```js
function makeUpperCase(str) {
  return str.toUpperCase();
}
```
