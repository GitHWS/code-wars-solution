# Opposite number

## Requirement

<p>Very simple, given an integer or a floating-point number, find its opposite.</p>

  1. 정수 또는 소수점을 가진 숫자의 반대를 반환해라

## Example

```js
1: -1
14: -14
-34: 34
```

<br>

### 첫번째 나의 시도

> why? 정수에 -만 붙여주면 양수는 음수가 되고 음수는 양수가 되기 때문에  
```js
function opposite(number) {
  return -number;
}
```
